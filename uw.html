Flashcards are the easiest way for me to retain information. At this point, I don't take notes for my classes anymore. It's all flashcards all the way, baby. I use the Anki software to create and host them. Anki also allows sharing decks publicly, which is what i've done with some of the courses i've completed, as below.

  • AP BIOLOGY. Actually this is a combined deck for both AP Bio and the Alberta Bio 20/30 curriculum. There's a lot of overlap between these two courses and I was taking them simultaneously, so I didn't bother separating the two at the time.
  • CHEM 120. For CHEM 120 only, not CHEM 120L!! (Although there is overlap between the finals, this deck does not provide comprehensive coverage of exam content for CHEM 123L.) See my review for the course here.
